# מאמן Revised Simplex — פלטפורמת תרגול

כלי תרגול אינטראקטיבי בעברית לאלגוריתם הסימפלקס המעודכן (Revised Simplex),
לפי סדר השלבים וההגדרות של הקורס "מודלים דטרמיניסטיים בחקר ביצועים".

## הפעלה

**האתר החי (מכל מכשיר):** https://amitsagy.github.io/simplex-revised-tutor/

או מקומית: אין התקנה ואין שרת — פשוט פתחו את `index.html` בדפדפן (לחיצה כפולה).

- `index.html` — המאמן: מזינים תרגיל, ופותרים אותו צעד-צעד.
- `calculator.html` — מחשבון דירוג מטריצות עצמאי (פעולות שורה אלמנטריות).

## הרעיון

המערכת **לא** מדריכה מראש — אתם מובילים:

1. **"מה השלב הבא?"** — בכל צומת אתם נזכרים בעצמכם מה השלב הבא באלגוריתם.
   רשימת האפשרויות מוסתרת; ריחוף/לחיצה חושפים אותה (ונרשם כרמז קל).
2. **זיהוי כל גודל** — לפני מילוי ערכים מצהירים איזה אובייקט בונים
   (B? cB? n̄q?...) ומה הממדים שלו. זה מה שמשנן את ההגדרות.
3. **מילוי ובדיקה** — ערכים מחושבים מוקלדים (עשרוני או שבר `a/b`) ונבדקים
   תא-תא. מטריצות B ו-N נבנות ב**בחירת עמודות** מ-(A|I) המקורית — הבחירה
   היא ההגדרה; ההעתקה היא רק אלגברה.
4. **רמזים מדורגים** — רמז קל ← רמז מפורט ← חשיפת תשובה. כל עזרה נרשמת,
   ובסוף התרגיל מוצג סיכום "במה נעזרת" כדי לדעת מה לחזק לפני המבחן.

עדכון B⁻¹ בשלב 5 — האלגברה לא חוסמת את הלמידה, בוחרים כמה להתעמק:

- הקלדה ישירה, **או** המחשבון המוטמע (פעולות שורה על [B⁻¹ | n̄q]) עם
  שלוש רמות נוחות: פעולות ידניות · **פיבוט בלחיצה על תא** (לחיצה אחת =
  הפיכה ל-1 ואיפוס העמודה) · **צעד אוטומטי / פתור עד הסוף** (כל צעד
  מתועד ביומן — רואים את הדרך).
- **"⚡ חשב עבורי (דירוג אוטומטי)"** — תשובה סופית בלחיצה אחת למי שלא
  רוצה להתעכב על האריתמטיקה. נרשם כ"חישוב אוטומטי" (בחירה לגיטימית),
  בנפרד מסיכום העזרה — זה לא נחשב חולשה בשינון.

במחשבון העצמאי (`calculator.html`) יש גם **מצב "מציאת הופכית"**: מזינים
B ריבועית, עובדים על [B | I] (ידנית/פיבוט-בלחיצה/אוטומטית), וכשהחצי
השמאלי הופך ל-I — B⁻¹ מוצגת אוטומטית.

עוד בפלטפורמה:

- **אזור חישוב (כפל מטריצות)** בכל שלב חישובי (xB, Z, yᵀ, rN, n̄q): שתי
  מטריצות (טעונות מנתוני השלב או מוזנות ידנית), מכפלה שאתם ממלאים ובודקים
  או נותנים לפלטפורמה לחשב, ול-rN — שלב חיסור נפרד (cN − yᵀN) כדי לראות
  את המינוס בעיניים. "השתמש בתוצאה כתשובה" מגיש דרך הבדיקה הרגילה.
- **כפתור "למה?"** אחרי כל תשובה נכונה — ריחוף/לחיצה מציגים את הנימוק
  המלא עם הערכים של השלב (לא נספר כעזרה).
- **פיצול cᵀ** מוצג בכל איטרציה: הווקטור המלא עם צביעת בסיסי/לא-בסיסי,
  לצד cB ו-cN.
- **"שאל את קלוד" 💬** — פאנל צד לשאלות חופשיות כשנתקעים. ההקשר המלא של
  התרגיל מצורף אוטומטית, בשלושה מסלולים:
  1. **"פתח בקלוד" (ברירת המחדל, חינם)** — נפתח טאב של claude.ai עם
     השאלה וההקשר כבר כתובים בצ'אט; נשאר רק לשלוח. דרוש רק חשבון
     claude.ai חינמי — בלי מפתח ובלי הגדרות.
  2. "העתק שאלה+הקשר" — להדבקה בכל צ'אט.
  3. מתקדם (מקופל): מפתח API אישי — התשובה מופיעה בתוך הפאנל.

### מפת דרכים למוצר ציבורי

האתר סטטי לחלוטין ועובד כמות-שהוא גם באירוח כמו GitHub Pages. אם הכלי
יופץ כמוצר אמיתי, השלב הבא לתשובות-בתוך-האפליקציה לכולם הוא שרת קטן
שמחזיק מפתח אחד של בעל האתר עם מכסת שימוש (אותה תבנית כמו ה-proxy של

**פרסום לאינטרנט כשמחליטים** (המאגר המקומי כבר מוכן, gh כבר מחובר):

```

gh api -X POST repos/{owner}/simplex-revised-tutor/pages -f "source[branch]=main" -f "source[path]=/"
```

שים לב: מאגר ציבורי = גם הקוד גלוי (אין בו סודות; מפתח ה-API נשמר רק
בדפדפן). ההתקדמות נשמרת מקומית בכל מכשיר בנפרד.

## מצבי תרגול

הפלטפורמה מכסה את חומר תרגולים 8–10, כל אחד בשיטת "הסטודנט מוביל" (שאלה →
הצהרת ממדים → מילוי ובדיקה תא-תא, רמזים מדורגים, "למה?"):

- **🔁 סימפלקס מעודכן (תרגול 8)** — פתרון בעיית `Max` צעד-צעד. כולל
  **🎲 תרגיל אקראי** (מספרים נוחים, 2–4 איטרציות; ~75% אופטימלי, ~25%
  לא-חסום), ו-**איטרציה בודדת מבסיס נתון** (סגנון ת"ב 8 ש2).
- **🔄 דואליות (תרגול 9)** — בונים את הבעיה הדואלית צעד-צעד לפי טבלת
  ההצמדה (כיוון → ממדים → מקדמי מטרה → אילוצים → תחומי משתנים), כולל
  צורות לא-סטנדרטיות (=, ≥, משתנה חופשי). **🎲 דואליות אקראית** זמינה.
- **⚙️ סימפלקס דואלי (תרגול 10)** — בוחרים קודם את המשתנה היוצא (ה-RHS
  השלילי ביותר), מבחן יחס לאורך השורה, פיבוט על כל הטבלה. **🎲 אקראי** זמין.
- **🎛️ ניתוח רגישות (תרגול 10)** — שינוי אגף ימין / מקדם בסיסי / מקדם
  לא-בסיסי / משתנה חדש / טווח שינוי מותר, מול הטבלה האופטימלית הנתונה.
- **🔍 תרגול שחזור (סגנון שאלה 3)** — נתונה הטבלה האופטימלית ומשחזרים
  אחורה את הבעיה: B⁻¹ מתחת לסרק → B=A, y משורת ה-Z → cᵀ=yᵀB, והשלמת ה-RHS.
- **🧭 תרגיל מודרך** — שרשרת שאלות המובילה מפתרון אחת הבעיות לפתרון השנייה
  (ת"ב 9 ש6: מפעל התכשיטים, דרך משתני סרק משלימים).
- **📖 כרטיסי קריאה (הוכחות)** — הוכחות הדואליות (חלשה/חזקה, משתנה חופשי,
  סימטריות) כטקסט מלא, עם כפתור "פתח בקלוד" לשאלות המשך.
- **📝 מצב מבחן** — חל על כל המצבים: בלי רמזים ובלי "חשב עבורי", שעון
  וספירת שגיאות, וציון בסוף (`max(0, 100 − 3×שגיאות)`).
- **מחירי צל** במסך הסיום של כל תרגיל אופטימלי.

## 📚 ספריית התרגילים

כפתור **"בחר תרגיל מהתרגולים"** במסך הפתיחה פותח בורר של כל התרגילים
המספריים מתרגולים 8–10 (וכל שאלות תרגילי הבית), מקובצים לפי מקור עם תג
סוג לכל תרגיל. כל תרגיל מריץ אוטומטית את מצב התרגול המתאים לו. התשובות
תואמות ערך-אחר-ערך את הפתרונות המודפסים בחומרי הקורס.

## היקף

**סימפלקס קדימה:** בעיות `Max Z = cᵀx` עם אילוצי `≤` ו-`b ≥ 0` (בסיס סרק
התחלתי ישים, בלי Big-M / שני שלבים), כולל התחלה מבסיס נתון באמצע ריצה.
**דואליות:** כל הצורות (Max/Min, ≤/=/≥, משתנה ≥0/חופשי/≤0). **סימפלקס
דואלי:** בעיות `Min` עם אילוצי `≥`. **רגישות:** שינויי b, c ומשתנים על
בסיס אופטימלי נתון. זיהוי עצירה כולל פתרון אופטימלי, לא-חסום, ומרובה.

טיפ דיוק: בערכים לא שלמים הזינו שבר (`1/3`) או לפחות 3 ספרות אחרי הנקודה.

## בדיקות

```
node test/run-tests.js
```

ללא תלויות. הבדיקות משחזרות ערך-אחר-ערך את שתי הדוגמאות הפתורות מחומרי
הקורס (דוגמת הכיתה — אופטימלי Z*=36; תרגיל בית 1א' — לא חסום), ומריצות את
מכונת המצבים מקצה לקצה. שתי הדוגמאות זמינות גם ככפתורי טעינה מהירה במסך
"תרגיל חדש".

## מבנה הקוד

```
js/parse.js          קלט טקסט -> מספר (עשרוני / שבר), פורמט תצוגה
js/answer-check.js   השוואת תשובות עם טולרנס לעיגול ידני
js/row-reduce.js     פעולות שורה אלמנטריות (משותף למנוע ולמחשבון!)
js/engine.js         מנוע הסימפלקס הטהור + היפוך מטריצה — התשובה הנכונה לכל שלב
js/duality.js        מנוע ההצמדה פרימל↔דואל (טבלת תרגול 9)
js/tableau.js        מנוע הסימפלקס הדואלי (טבלה מלאה) + עזרי רגישות
js/generator.js      מחוללי תרגילים אקראיים (סימפלקס / דואליות / ס' דואלי)
js/session.js        מכונת המצבים לכל המצבים: recall -> dims -> fill, איטרציות
js/exercises.js      ספריית התרגילים (נתונים) מתרגולים 8–10
js/*-ui.js, main.js  שכבת התצוגה (ווידג'טים + הוויזארד + בורר הספרייה)
test/                הרצת בדיקות + fixtures מחומרי הקורס
```

הקבצים הלא-ויזואליים רצים גם בדפדפן (`window.Simplex.*`) וגם ב-Node
(`module.exports`) — אותו קוד בדיוק נבדק ומורץ.
